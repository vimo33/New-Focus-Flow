# Ray Edwards Copywriting

Write persuasive sales copy using Ray Edwards PASTOR framework.

## Description

This skill applies Ray Edwards copywriting methodology - a modern approach that combines classic direct response with ethical persuasion.

## The PASTOR Framework

### P - Problem
Identify and articulate the readers pain point.

### A - Amplify
Make the problem feel urgent and real.

### S - Story/Solution
Tell a story and present your solution.

### T - Transformation
Show the before and after.

### O - Offer
Present what they get and the value.

### R - Response
Clear call to action.

## Usage

When asked to write copy, I will:

1. Identify the core problem
2. Amplify the pain points
3. Tell a relatable story
4. Show the transformation possible
5. Make an irresistible offer
6. Create clear next steps

## Key Principles

- Ethical persuasion over manipulation
- Story-driven selling
- Transformation focus
- Clear value articulation
- Strong guarantees
