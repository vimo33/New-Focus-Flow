# Component Inventory (Build once, reuse everywhere)

## Shell
- AppShell (grid + noise)
- TopBar (title, live indicator, user)
- BottomNav (existing) + badges

## Primitives
- GlassCard (default/elevated/critical)
- MetricTile
- StatusChip (Active/Draft/Planned/Queue/Blocked/Risk)
- Button (primary cyan, secondary ghost, danger)
- ProgressStepper (workflow timeline)
- GaugeRing (score/health)
- RiskMatrix mini
- Callout (info/warn/risk)
- ApprovalBar (inline YES/NO/Approve/Edit)
- ExpandableText (3-line clamp + expand)
- Drawer / Popover (evidence, long text)

## Voice
- VoiceBar (text input + mic)
- VoiceOverlay (live transcript)
- VoiceStatePill (Listening/Thinking/Acting/Waiting approval)

## Power features
- CommandPalette (Cmd+K)
- DataViewer (PDF embed panel)
- GraphView (network)
