Usage Rules & Semantics

Accessibility Checklist

check_circle Text contrast ratio > 4.5:1 on dark backgrounds
check_circle Interactive targets min 44px height
check_circle Semantic color redundancy (icon + color)
DO
  • PURPLE Primary actions, AI features
  • GREEN Success, growth, positive delta
  • RED Destructive, errors, termination
DON'T
  • Use Red for simple warnings
  • Use Purple for decorative backgrounds only
  • Mix Green with brand Blue indiscriminately
How to extend

"When adding new patterns, prioritize information density without sacrificing clarity. Glass panels should be used to separate contexts, not just for decoration. Ambient lighting indicates system state (Blue=Idle, Purple=Thinking, Red=Critical)."
